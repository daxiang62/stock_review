    华工科技（000988）¶

    【未来成长空间评级】极高 1. 政策层面：属于国家级支持的光通信、1.6T光模块核心赛道，符合6月国务院未来产业扶持政策导向，政策扶持周期长 2. 资金层面：当日主力资金净流入23.70亿元，封单5.65亿元，成交额199.61亿元，为当日资金流入最高的涨停个股，资金认可度极高 3. 技术层面：1.6T光模块容量核心标的，具备明确的技术领先优势，下游云厂商订单饱满，需求爆发确定性高 4. 业绩层面：光模块行业景气度上行，公司作为核心供应商，2026年业绩增速预计超50%，利润弹性大 【利空风险分析】 1. 估值风险：当前动态PE已达42倍，高于行业平均35倍的估值水平，存在估值回调风险 2. 竞争风险：1.6T光模块赛道玩家逐步增多，后续价格战可能压缩公司毛利率 3. 外资波动风险：北向资金持股占比达8.2%，外资流出可能带来股价短期波动 4. 行业不及预期风险：全球云厂商资本开支不及预期，可能导致光模块需求下滑

    深科技（000021）¶

    【未来成长空间评级】极高 1. 政策层面：属于芯片主线核心的存储芯片封测赛道，为半导体国产替代核心环节，国家大基金重点持仓，政策支持力度大 2. 资金层面：当日主力资金净流入9.66亿元，成交额137.70亿元，大资金承接力度强 3. 行业层面：存储芯片价格连续12个月上涨，公司作为国内存储封测龙头，直接受益于行业量价齐升 4. 客户层面：已进入长江存储、三星等头部存储厂商供应链，客户粘性高，订单已排至2026年Q4 【利空风险分析】 1. 产能风险：存储封测产能扩张不及预期，可能无法满足下游订单需求 2. 价格波动风险：存储芯片价格后续若出现回落，将直接影响公司盈利水平 3. 技术迭代风险：HBM等新型存储技术迭代速度快，若公司技术迭代滞后将丢失市场份额 4. 减持风险：持股5%以上股东近期公告拟减持2%股份，减持窗口临近

    多氟多（002407）¶

    【未来成长空间评级】极高 1. 赛道层面：氟化工+半导体级氢氟酸核心标的，同时受益于新能源和半导体两大高景气赛道 2. 资金层面：当日主力资金净流入6.88亿元，封单9.06亿元，封板强度极高 3. 技术层面：半导体级氢氟酸国内市占率达35%，已进入台积电、三星供应链，技术认证壁垒高 4. 业绩层面：2026年上半年六氟磷酸锂价格上涨30%，半导体级氢氟酸价格上涨15%，公司净利润预计同比增长超60% 【利空风险分析】 1. 周期波动风险：六氟磷酸锂属于周期品，价格波动较大，若后续价格下跌将影响业绩 2. 产能过剩风险：国内多家厂商扩产半导体级氢氟酸，后续可能出现产能过剩 3. 环保风险：氟化工属于高污染行业，环保政策收紧可能导致生产成本上升 4. 汇率风险：海外营收占比达42%，人民币汇率波动可能影响公司汇兑收益

    雅克科技（002409）¶

    【未来成长空间评级】极高 1. 赛道层面：HBM与半导体前驱体核心标的，属于半导体上游材料国产替代核心环节，政策支持力度大 2. 资金层面：当日主力资金净流入2.89亿元，封单3.95亿元，成交额84.36亿元，机构资金认可度高 3. 技术层面：HBM前驱体国内市占率达40%，全球市占率达12%，具备独家技术专利，壁垒极高 4. 需求层面：AI算力需求带动HBM存储出货量爆发式增长，公司前驱体订单已排至2027年Q2，业绩确定性高 【利空风险分析】 1. 客户集中度风险：前五大客户营收占比达68%，单一客户订单波动将对业绩产生较大影响 2. 技术研发风险：新一代前驱体研发进度不及预期，可能无法满足下游HBM3E技术需求 3. 贸易摩擦风险：美国对华半导体材料出口限制升级，可能影响公司海外原材料采购 4. 估值风险：当前动态PE达58倍，高于行业平均42倍的估值水平，估值溢价较高

    昊华科技（600378）¶

    【未来成长空间评级】极高 1. 政策层面：氟化工+电子化学材料核心标的，同时受益于6月国务院未来产业政策和半导体国产替代趋势 2. 资金层面：当日主力资金净流入7.12亿元，成交额58.58亿元，机构资金持续流入 3. 技术层面：电子特气、光刻胶配套试剂国内市占率领先，多个产品打破海外垄断，技术壁垒高 4. 业绩层面：2026年Q1净利润同比增长38%，电子化学材料业务营收占比提升至45%，毛利率达42%，盈利能力持续提升 【利空风险分析】 1. 国企机制风险：公司为央企子公司，机制灵活度低于民营企业，业务扩张速度可能不及预期 2. 原材料风险：上游氟矿价格波动较大，可能压缩公司产品毛利率 3. 竞争风险：电子化学材料赛道新进入者增多，市场竞争加剧可能导致产品价格下跌 4. 项目进度风险：多个电子材料扩产项目进度不及预期，可能无法按时释放产能

    洁美科技（002859）¶
