# 关于

## 项目介绍

股票分析看板是一个自动化的股票分析系统，利用大模型技术对每日涨停、炸板、断板等股票进行深度分析。

## 核心功能

- **涨停板分析**: 分析当日涨停股票的走势、板块分布、连板情况
- **炸板分析**: 分析炸板股票的原因和后续走势
- **断板分析**: 分析断板股票的特征和风险
- **每日盘面数据**: 统计市场涨跌、量能、热点板块等数据

## 技术栈

- **数据分析**: Pandas, AkShare
- **大模型分析**: 豆包大模型 (Volc Engine)
- **报告生成**: Markdown
- **看板展示**: MkDocs + Material 主题

## 目录结构

```
stock_review/
├── analyzers/          # 分析器模块
├── reporters/         # 报告生成器
├── docs/              # MkDocs 文档
├── reports/           # 生成的报告
├── utils/             # 工具函数
└── daily_analyze_stock.py  # 每日分析脚本
```

## 使用说明

### 运行每日分析

```bash
python daily_analyze_stock.py
```

### 构建看板

```bash
mkdocs serve
```
